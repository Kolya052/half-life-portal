# Half-Life-Game
# 🔥 HALF-LIFE: SHATTERED REICH

> *Альтернативная история. Нацисты выиграли войну. Технологии из будущего. Ты — последняя надежда.*
---

## 🎮 ОБ ИГРЕ

**Half-Life: Shattered Reich** — это альтернативная история во вселенной Half-Life. 

Что если бы **нацисты выиграли Вторую мировую**, используя технологии из будущего? Что если бы **Гордон Фримен** был не физиком, а **инженером Рейха**, работающим на заводе Waffenfabrik IV?

**Твоя цель:** Спасти дочь Алекс из Объекта 7 и остановить безумный план Фюрера, который хочет открыть портал в мир Зен.

## 🎯 СЮЖЕТ

**Глава 1: Прибытие** — Ты на вокзале Stadt 19. Атака Сопротивления. Первая встреча с G-Man.

**Глава 2: Подземелье** — База Сопротивления. Встреча с Кляйнером. Получение H.E.X. костюма.

**Глава 3: Завод** — Waffenfabrik IV. Клоны Рейха. Встреча с Криматором.

**Глава 4: Объект 7** — Финальный штурм. Босс — Железный Волк. Спасение Алекс.

## 🎮 УПРАВЛЕНИЕ

| Клавиша | Действие |
|---------|----------|
| **WASD** | Движение |
| **Пробел** | Прыжок |
| **Ctrl** | Присесть |
| **Q / E** | Наклоны |
| **1-4** | Сменить оружие |
| **ЛКМ** | Стрельба / удар / поднять (грави-пушка) |
| **ПКМ** | Бросить (грави-пушка) |
| **E** | Поднять банку / взаимодействие |
| **F** | Кинуть банку |
| **\** | Консоль |
| **Ъ / B** | Режим Бога |
